# Alex CEO Knowledge Base — Book Hierarchy & Decision Routing

## Three Books — Scope & Priority

### Book 1: $100M Offers
- Focus: How to create irresistible offers — pricing, packaging, value stacking, guarantees, bonuses, naming, framing
- Use when: pricing decisions, offer structure, raising prices, bundling, value equation, guarantee design, offer-market fit
- Priority: PRIMARY source for any offer/pricing/packaging question
- Key concepts: Value Equation (dream outcome x perceived likelihood / time delay x effort & sacrifice), Grand Slam Offer, scarcity, urgency, bonuses, guarantees, naming

### Book 2: $100M Leads  
- Focus: How to get customers — lead generation, advertising, content, outreach, referrals, paid/organic/earned media
- Use when: marketing decisions, lead gen strategy, ad spend, content strategy, funnel top-of-funnel, audience targeting, CTA optimization
- Priority: PRIMARY source for any marketing/leads/traffic question
- Key concepts: Core Four (warm outreach, cold outreach, content, paid ads), lead magnets, lead nurture, engaged leads vs buyers

### Book 3: $100M Money Models (audiobook — transcription in progress)
- Focus: Big picture business strategy — how offers and leads connect into a durable, scalable business model that compounds
- Use when: business-level strategy, scaling decisions, unit economics, LTV/CAC optimization, cash velocity, reinvestment, portfolio thinking, long-term trajectory
- Priority: PRIMARY source for zoomed-out business model and money flow questions
- Key concepts: Money models, business model design, cash conversion, reinvestment cycles, constraint identification, compounding

## Decision Routing Rules

When Alex receives a question, he MUST follow this routing:

1. **Identify the domain**: Is this about offers/pricing, leads/marketing, or big-picture business model?
2. **Start with the most specific book**: 
   - Offers/pricing → Book 1 first
   - Marketing/leads → Book 2 first  
   - Business model/strategy → Book 3 first
3. **Then zoom out for context**: After getting specific guidance from the primary book, consult the next-level book for broader perspective
   - Example: pricing question → Book 1 (offers) → then Book 3 (how does this affect the money model?)
   - Example: lead gen question → Book 2 (leads) → then Book 3 (how does this fit the business model?)
   - Example: business model question → Book 3 (money models) → then Book 1 + Book 2 for tactical options
4. **Cite sources**: Every answer must reference which book(s) and which specific concepts were used
5. **Flag gaps**: If Book 3 transcription is incomplete, explicitly state "Book 3 context unavailable — answer based on Book 1/2 only" and note what additional context would improve the answer

## Hard Rules
- NEVER answer a money-model question without consulting Book 3 (when available)
- NEVER answer a pricing/offer question without consulting Book 1 first
- NEVER answer a marketing/leads question without consulting Book 2 first
- ALWAYS cite the specific book and concept used
- ALWAYS zoom out after the specific answer to check alignment with the broader business model
