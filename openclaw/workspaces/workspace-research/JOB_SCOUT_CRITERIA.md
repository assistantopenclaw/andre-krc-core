# JOB_SCOUT_STANDING_RULES.md (v3.1)

## Primary Objective
Identify remote U.S. roles landable within 30 days that match Andre's proof and prioritize fit, traction, and autonomy over prestige.

## Non-Negotiable Output Rules
1. **Direct Links Only**: A role MUST have a specific direct application URL or direct company career page for that specific role.
2. **Invalid Links**: No homepages, search results, category pages, or "search X site" instructions.
3. **Strict Eligibility**: Roles must have identified company, title, specific URL, score 70+, and pass all dealbreaker filters (support/phone/meeting-heavy).
4. **No Guessed Data**: Compensation must be exact listed amount/range or "Not listed". No estimates.
5. **Confidential Listings**: Heavily penalize; only include if unusually strong and direct URL exists.

## Sweep Status & Sources
- **COMPLETE** only if ALL 16 groups searched (Indeed, LinkedIn, Glassdoor, ZipRecruiter, Built In, Wellfound, WWR, Remote OK, Remote.co, Remotive, YC, Working In Content, Greenhouse, Lever, Ashby, Direct).
- **PARTIAL** if any are skipped. Explicitly list skipped sources and exact reasons.
- Mainstream sources (Indeed/LinkedIn/Glassdoor) are MANDATORY; remote boards are NOT substitutes.

## Scoring Logic (1-100)
- **25 pts**: Fit to proof (100k growth, 27M views, 5.6k leads).
- **20 pts**: Traction probability (Speed to interview).
- **20 pts**: Autonomy / Low meeting load (Async preference).
- **15 pts**: AI-compressibility (Writing/research focus).
- **10 pts**: Compensation quality.
- **5 pts**: Contract flexibility / Easy exit.
- **5 pts**: Company quality / Low risk.

## Search Procedure & Bias
- **60%**: Growth Creative / Performance Copy.
- **30%**: Demand Gen / Lifecycle (Messaging-heavy).
- **10%**: SEO / Content Ops / Systems.
- **Procedure**: Search -> Term Sweep (30 terms) -> Candidate Pool (25-40) -> Validation -> Scoring -> Reporting.

## Report Format
### TOP HEADER
- Sweep status: Complete or Partial
- Valid applyable roles found: X
- Discovery leads requiring link resolution: Y
- Required sources searched: List
- Required sources skipped: List with exact reasons

### SECTION A: APPLYABLE ROLES
(Rank, Score, Company, Title, Specific URL, Source, Freshness, Compensation, Fit Analysis, Traction Read, Autonomy/AI Read, Weakness, Verdict)

### SECTION B: DISCOVERY LEADS (Max 5)
(Unusually strong leads without direct URLs - labeled NOT READY TO APPLY)
