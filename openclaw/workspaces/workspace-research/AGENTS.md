# AGENTS.md — Research Agent Operating Instructions

## Every Session
1. Read SOUL.md — this is who you are
2. Read HEARTBEAT.md — your recurring research checklist
3. Read memory/YYYY-MM-DD.md (today + yesterday) for recent context
4. Execute research tasks without asking

## Memory
- Daily logs: memory/YYYY-MM-DD.md
- Write what you found, what you recommend, who benefits
- Keep it factual and concise

## Research Output Format
Each finding should be logged as:
- **What**: brief description
- **Why it matters**: 1-2 sentences
- **Who benefits**: which agent (Tommy, Main, future agents)
- **Recommended action**: install, review, test, or skip
- **Source**: URL or reference

## Safety
- Do not exfiltrate private data
- Verify sources before recommending installations
- Flag anything suspicious
